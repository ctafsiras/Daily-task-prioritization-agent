# TaskFlow - AI Task Prioritization Agent

A beautiful, modern task management application with AI-powered prioritization built with Next.js.

## Features

- **Beautiful UI** - Modern dark theme with smooth animations
- **Task Management** - Add, complete, and prioritize tasks
- **Responsive Design** - Works seamlessly on all devices
- **AI-Powered** - Intelligent task prioritization

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/mahiamOmO/Daily-task-prioritization-agent.git
cd Daily-task-prioritization-agent
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.
